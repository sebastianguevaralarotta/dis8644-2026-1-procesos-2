Tuvimos una introducción rápida pero completa sobre la evolución de la música.

En el canto gregoriano se utilizaba el tetragrama. Fue inventado por el monje Guido de Arezzo para representar la altura de las notas musicales de forma más precisa y a intervalos de tercera . El pentagrama surge en el siglo XV la cual se agregó para ampliar el rango de notas musicales que se podían escribir e interpretar.
